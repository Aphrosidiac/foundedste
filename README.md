# FOUNDED — Dark Art Streetwear

An award-worthy, scroll-driven editorial website for FOUNDED, a Malaysian dark art streetwear brand. Built with a focus on cinematic aesthetics, immersive animations, and editorial-grade layout design.

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** — scroll-driven animations, parallax, clip-path reveals
- **Lenis** — buttery smooth scrolling

## Features

- **Preloader** — Cinematic brand intro with progress counter and slide-up reveal
- **Custom cursor** — Mix-blend-difference cursor with hover scaling (desktop)
- **Film grain overlay** — Animated noise texture for raw, textured feel
- **Hero** — Text scramble/glitch effect resolving into display type, parallax background
- **Scroll-reactive marquee** — Dual-direction ticker that accelerates with scroll velocity, mixed typography styles
- **Collection grid** — Clean 3-column product grid with a 2-col hero break, scroll parallax on images
- **Featured section** — Sticky split-screen with clip-path circle reveal, 3-part scroll narrative, animated counters
- **Horizontal gallery** — Vertical scroll drives horizontal image movement with staggered entry
- **Lookbook** — Cursor-reactive 3D tilt on images with spring physics, scroll parallax
- **Manifesto** — Scroll-locked viewport with line-by-line text reveal and blur-in effect
- **Section dividers** — Animated horizontal lines that draw across the viewport
- **Footer** — Large typographic CTA with stroke text

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
